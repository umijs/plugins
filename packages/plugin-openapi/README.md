# @umijs/plugin-openapi

> @umijs/plugin-openapi.

See our website [@umijs/plugin-openapi](https://umijs.org/plugins/plugin-openapi) for more information.

## Install

Using npm:

```bash
$ npm install --save-dev @umijs/plugin-openapi
```

or using yarn:

```bash
$ yarn add @umijs/plugin-openapi --dev
```
