# @umijs/plugin-stylus

> @umijs/plugin-stylus.

See our website [@umijs/plugin-stylus](https://umijs.org/plugins/plugin-stylus) for more information.

## Install

Using npm:

```bash
$ npm install --save-dev @umijs/plugin-stylus
```

or using yarn:

```bash
$ yarn add @umijs/plugin-stylus --dev
```
