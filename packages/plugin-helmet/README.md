# @umijs/plugin-helmet

> @umijs/plugin-ssr-helmet.

See our website [@umijs/plugin-helmet](https://umijs.org/plugins/plugin-helmet) for more information.

## Install

Using npm:

```bash
$ npm install --save-dev @umijs/plugin-helmet
```

or using yarn:

```bash
$ yarn add @umijs/plugin-helmet --dev
```
