See our website [@umijs/plugin-access-layout](https://umijs.org/plugins/plugin-access-layout) for more information.

## Install

Using npm:

```bash
$ npm install --save-dev @umijs/plugin-access-layout
```

or using yarn:

```bash
$ yarn add @umijs/plugin-access-layout --dev
```
